# SnipVault

SnipVault is a desktop app that allows developers to manage their code snippets in one place.
This is a project made for learning purposes in order to dive into Tauri, React and TypeScript.

This is still in very early development.

## Stack

- Tauri
- React + TypeScript
- Tailwind CSS

## Features

- Create, delete and view your snippet.
- SQLite database to store your snippets.

## Roadmap

- Create the main snippet view.
- MonacoEditor for a smooth code editing experience.
- Snippet collections to organize your snippets.
- Tags and languages filters to easily find your snippets.

## Developpement

```bash
npm install
npm run tauri dev
```
